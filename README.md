# Convert RGB color to HEX

## Installation

Package is distributed via [npm](https://www.npmjs.com/):

```javascript
npm i @defo550/rgb-to-hex
```

## Usage

```javascript
const rgbToHex = require('@defo550/rgb-to-hex');

rgbToHex(246, 141, 100);
// 'f68d64'

rgbToHex('rgb(58 81 113)');
// '3a5171'
```
